# Таблиці HTML та CSS Стилізація

Навчальний проект для вивчення HTML таблиць та їх оформлення за допомогою CSS.

## 📋 Опис

Проект демонструє створення та стилізацію HTML таблиць з використанням:

- **colspan** та **rowspan** для об'єднання комірок

## 🎨 Структура проекту

```
lesson_6/
├── index.html          # Головна сторінка з таблицею Mondrian
├── styles/
│   ├── reset.css      # CSS Reset для нормалізації стилів
│   └── style.css      # Основні стилі для всіх таблиць
└── table/
    ├── table1.html    # Приклад таблиці 1 з об'єднаними комірками
    └── table2.html    # Приклад таблиці 2 з вкладеною таблицею
```

## 🚀 Як використовувати

1. **Відкрити головну сторінку:**
   - Відкрийте `index.html` у браузері

2. **Навігація:**
   - Натисніть кнопку **Table1** для перегляду першої таблиці
   - Натисніть кнопку **Table2** для перегляду другої таблиці

3. **Редагування:**
   - Змініть вміст HTML файлів для додавання нових рядків/комірок
   - Модифікуйте `styles/style.css` для зміни зовнішнього вигляду

## 🎯 Ключові особливості

### Таблиця Mondrian (index.html)

- Вкладена таблиця з багатокольоровими комірками
- Комірки з різними розмірами та об'єднаннями
- Стильні навігаційні кнопки з ефектом наведення

### Стилізація

- **Кольори:** жовтий, червоний, синій, чорний фон
- **Фіксована ширина:** для окремих комірок (25px, 75px, 100px, 200px)
- **Висота рядків:** 20px, 30px, 50px, 100px, 200px
- **Кнопки навігації**

## 📐 Класи CSS

### Висота

- `.h20` - 20px
- `.h30` - 30px
- `.h50` - 50px
- `.h100` - 100px
- `.h200` - 200px

### Ширина

- `.w25` - 25px
- `.w75` - 75px
- `.w100` - 100px
- `.w200` - 200px

### Кольори

- `.yellow` - жовтий фон
- `.red` - червоний фон
- `.blue` - синій фон
- `.black` - чорний фон з білим текстом

### Таблиці

- `.table1` - 75% ширини
- `.table2` - 90% ширини
- `.table2-1` - вкладена таблиця 80% ширини
- `.mondriana` - основна таблиця 50% ширини з фіксованим макетом
